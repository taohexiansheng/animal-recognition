# 动物识别助手

这是一个基于百度AI开放平台的动物识别工具。用户输入动物图片的URL，程序调用百度动物识别API，返回图片中的动物名称和置信度。

## 功能
- 输入动物图片的网址
- 返回识别结果（动物名称 + 匹配度）,根据概率由大到小排序
- 支持多种动物识别（如猫、狗、企鹅等）

## 技术栈
- Java
- 百度AI动物识别API
- Git / GitHub

## 如何运行
1. 在百度智能云控制台获取 API Key 和 Secret Key
2. 替换代码中的密钥
3. 运行 `AnimalDemo.java`，输入图片URL

## 示例
输入：`https://example.com/cat.jpg`  
输出：
```json
{
  "result": [
    {"score": 0.98, "name": "家猫"}
  ]
}
